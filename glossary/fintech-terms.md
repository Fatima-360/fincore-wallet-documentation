This glossary provides standardized definitions for the EuroFlow platform to ensure consistency across engineering, legal, and customer support documentation.

| Term | Definition | Context |
| :--- | :--- | :--- |
| **EU VAT Directive** | The legal framework governing Value Added Tax on digital services within the European Union. | Compliance / Legal |
| **VIES** | (VAT Information Exchange System) An electronic means of validating VAT-registration numbers of entities registered in the EU. | Technical / API |
| **MOSS** | (Mini One Stop Shop) A system that allows taxable persons to account for the VAT due on digital services in one single EU Member State. | Finance / Reporting |
| **SaaS** | (Software as a Service) A software licensing and delivery model in which software is licensed on a subscription basis. | Business Model |
| **Reconciliation** | The process of ensuring that two sets of records (e.g., bank statements and internal ledgers) are in agreement. | Finance / Audit |
| **Docs-as-Code** | An approach where documentation is treated like software code, using tools like Git, Markdown, and automated builds. | Engineering Workflow |
